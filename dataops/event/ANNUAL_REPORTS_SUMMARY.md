## Report Structure

### FY2025 Reports (Comprehensive)

Each report includes:
1. **CEO Letter**: Addresses FY2025 performance and NRNT episode
2. **Financial Highlights**: Full income statement summary
3. **Business Overview**: Platform capabilities, competitive positioning
4. **FY2025 Achievements**: Quarterly milestones and wins
5. **The NRNT Episode**: How each company responded to market noise
6. **Financial Performance**: Revenue analysis, unit economics
7. **Competitive Dynamics**: vs. specific competitors
8. **Strategic Priorities FY2026**: Forward-looking initiatives
9. **Risk Factors**: Business and market risks
10. **Outlook**: Guidance and long-term vision
11. **Contact Information**: IR, website, stock info

**Average Length**: ~250-300 lines per report

### FY2024 Reports (Concise)

Each report includes:
1. **CEO Letter**: FY2024 summary, forward-looking to FY2025
2. **Financial Highlights**: Key metrics
3. **Business Overview**: Positioning
4. **Achievements**: Major milestones
5. **Outlook**: FY2025 guidance

**Average Length**: ~60-75 lines per report

**Special**: NRNT FY2024 report longer (217 lines) to show warning signs

### Integration with Pipeline

**To add to pipeline**:
```sql
-- Upload annual reports to stage
PUT file:///annual_reports_2024/*.md @DOCUMENT_AI.ANNUAL_REPORTS/2024/;
PUT file:///annual_reports_2025/*.md @DOCUMENT_AI.ANNUAL_REPORTS/2025/;

-- Extract data with AI
CREATE TABLE ANNUAL_REPORTS_EXTRACTED AS
SELECT 
    RELATIVE_PATH,
    AI_EXTRACT(file_content, 'Extract: Company, FY Year, Revenue, Growth, Profit, CEO Name, Key Events') AS extracted_data
FROM @DOCUMENT_AI.ANNUAL_REPORTS;

-- Add to search service
CREATE CORTEX SEARCH SERVICE ANNUAL_REPORTS_SEARCH
ON full_text
ATTRIBUTES company, fy_year, revenue, ceo
AS (SELECT * FROM ANNUAL_REPORTS_EXTRACTED);
```

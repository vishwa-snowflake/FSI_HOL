## Advanced AI Capabilities Demonstrated

### Vision AI (Images)
```sql
-- Classify social media images
SELECT 
    SNOWFLAKE.CORTEX.AI_CLASSIFY(
        TO_FILE('@SOCIAL_MEDIA_IMAGES/' || RELATIVE_PATH),
        ['Product Success', 'Crisis', 'Executive', 'Personal Tragedy']
    ) AS category
FROM DIRECTORY(@DOCUMENT_AI.SOCIAL_MEDIA_IMAGES);

-- Filter crisis images
WHERE SNOWFLAKE.CORTEX.AI_FILTER(
    TO_FILE('@SOCIAL_MEDIA_IMAGES/' || filename),
    'This image shows product waste or crisis'
) = TRUE;
```

### Multi-Language AI (Text)
```sql
-- Translate Chinese posts
SELECT 
    SNOWFLAKE.CORTEX.AI_TRANSLATE(TEXT, 'zh', 'en') AS english,
    SNOWFLAKE.CORTEX.AI_SENTIMENT(translated) AS sentiment
FROM SOCIAL_MEDIA_NRNT
WHERE LANGUAGE = 'zh';

-- Aggregate across languages
SELECT 
    SNOWFLAKE.CORTEX.AI_AGG(
        AI_TRANSLATE(TEXT, LANGUAGE, 'en'),
        'What were the main NRNT concerns globally?'
    )
FROM SOCIAL_MEDIA_NRNT;
```

### Audio AI (Transcription)
```sql
-- Transcribe CEO interview
SELECT 
    SNOWFLAKE.CORTEX.AI_TRANSCRIBE(
        TO_FILE('@INTERVIEWS/CEO_Interview.mp3'),
        {'LANGUAGE': 'en', 'INCLUDE_TIMESTAMPS': TRUE}
    ) AS transcript;

-- Extract lessons from interview
SELECT 
    AI_EXTRACT(transcript, 'What lessons does the CEO mention learning?')
FROM transcribed_interviews;
```

### Geospatial AI (Location)
```sql
-- Map crisis spread
SELECT 
    COUNTRY,
    ST_MAKEPOINT(AVG(LONGITUDE), AVG(LATITUDE)) AS center,
    AVG(SENTIMENT) AS sentiment
FROM SOCIAL_MEDIA_NRNT
GROUP BY COUNTRY;

-- Find posts within 100km of location
WHERE ST_DISTANCE(
    ST_MAKEPOINT(LONGITUDE, LATITUDE),
    ST_MAKEPOINT(-122.4194, 37.7749)  -- San Francisco
) <= 100000;
```

# Layout Variation Strategy

## 4 Distinct Layout Styles for 8 Companies

### Style 1: Classic Professional (SNOW, DFLX)
- **Companies**: SNOW (market leader), DFLX (established platform)
- **Layout**: Current 2-column with left executive summary, right metrics/charts
- **Header**: Split (name left, ticker right)
- **Metrics**: 4 columns horizontal
- **Feel**: Professional, corporate, trustworthy

### Style 2: Bold Centered (NRNT, VLTA)
- **Companies**: NRNT (disruptive startup), VLTA (AI innovator)
- **Layout**: Centered header, 2x2 metrics grid, single column content
- **Header**: Centered with large company name
- **Metrics**: 2x2 grid with larger numbers
- **Feel**: Bold, modern, innovative

### Style 3: Minimal Technical (ICBG, QRYQ)
- **Companies**: ICBG (open source), QRYQ (price-performance)
- **Layout**: Compact header, inline metrics, wide content
- **Header**: Compact left-aligned
- **Metrics**: 2 large + 2 small asymmetric
- **Feel**: Clean, technical, efficient

### Style 4: Structured Grid (STRM, CTLG)
- **Companies**: STRM (real-time data), CTLG (governance)
- **Layout**: Traditional grid, 3-column content, organized sections
- **Header**: Traditional split with underline
- **Metrics**: Vertical stacked pairs
- **Feel**: Organized, structured, systematic

## Implementation Plan

1. Create 4 HTML template functions
2. Route each company to appropriate template
3. Maintain company-specific colors and data
4. Generate all 8 reports with varied layouts

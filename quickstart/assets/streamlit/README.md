# Streamlit Application

## StockOne Agent

This directory contains the StockOne Streamlit application - an AI-powered financial assistant using Snowflake Cortex Agents REST API.

### Files

- `app.py` - Main Streamlit application
- `styles.css` - Custom CSS styling
- `environment.yml` - Python dependencies
- `config.toml` - Streamlit configuration
- `Snowflake_dots.png` - Logo/branding

### Features

- ✅ Cortex Agents REST API integration
- ✅ Feedback API (thumbs up/down)
- ✅ Tool toggles (enable/disable search services)
- ✅ 3 LLM options (Claude, Mistral, Llama)
- ✅ Interactive visualizations
- ✅ Debug mode

### Deployment

This app is automatically deployed by `04_deploy_streamlit.sql`.

### Access

After deployment:
1. Navigate to: **AI & ML Studio** → **Streamlit**
2. Click: **STOCKONE_AGENT**
3. Start asking questions!

### Try

- "What is the latest Snowflake stock price?"
- "Show me revenue trend over last 4 quarters"
- "What did analysts say in latest earnings call?"
- "Compare sentiment across all companies"
